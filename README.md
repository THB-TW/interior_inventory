# 室內裝潢剩料管理系統 — 施工進度追蹤 + 智能叫料推薦

## 🚀 系統功能

**針對室內裝潢業設計，完整解決「剩料管理」與「案件進度追蹤」的痛點。**

### 📦 智慧庫存管理（剩料再利用）
- **入庫／出庫 / 報廢** 三大庫存異動，操作簡單、記錄完整
- **「案子用料」自動出庫**：從訂單建立時，自動從倉庫扣除對應材料
- **「結餘料處理」 自動入庫**：施工結束，將剩餘材料自動存回倉庫
- **自動生成「補料清單」**：系統比對「案件需求」與「庫存現況」，自動列出不足的材料，一鍵叫料
- **剩料追蹤顯示**：訂單中清楚標示哪些材料是「舊料再利用」，減少浪費

### 📋 施工進度追蹤（案件狀態）
- **案件狀態可視化**：一頁總覽所有案件，即時掌握進度（尚未排程、工班確認、施工中、驗收中、已完成）
- **進度自動跳轉**：
  - 建立訂單 → 自動為「尚未排程」狀態
  - 建立完畢 → 自動「派工排程」
  - 工班確認 → 自動進入「施工中」
- **案件曆視圖**：日曆模式查看所有排程，輕鬆安排檔期
- **工班協作**：工班可透過手機查看自己的案件，確認排程並回報進度

### 📱 LINE 智能通知（工班協作）
- **排程通知**：工班確認排程後，立即透過 LINE 通知「開始工作」
- **結案通知**：完工後立即傳送「结案通知」給老闆
- **狀態通知**：任何進度變更都會主動通知老闆

### 👥 多角色系統（老闆／工班）
- **老闆視角**：庫存總覽、訂單管理、案件進度、材料採購、工班調度
- **工班視角**：我的排程、案件進度回報、材料清單、手機操作
